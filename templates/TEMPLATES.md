# Templates bundle

Ten templates the site has promised. Each one is ready to paste into a Google Doc or Sheet. After pasting, share the doc as **Anyone with the link → Viewer**, copy the share URL, and we'll wire it into the matching `/resources/[name]/` page on the site.

Format note: items here are written in markdown for readability. To turn into a Google Doc, paste into a new Doc and apply Heading styles to the `#` and `##` lines. Tables paste cleanly into Sheets if copied through the rendered preview.

## 2. Carpool spreadsheet

**File:** `carpool-rotation.gsheet`

**Audience:** four-family carpool, recurring practice + game schedule.

**Body to paste into a Sheet (rendered as a table):**

| Date | Day | Event | Time | Driver out | Driver back | Notes |
|------|-----|-------|------|------------|-------------|-------|
| 9/3  | Tue | Practice | 5:30pm | [Name] | [Name] | |
| 9/5  | Thu | Practice | 5:30pm | [Name] | [Name] | |
| 9/7  | Sat | Game vs Eagles | 9:00am | [Name] | [Name] | Field 4 |

**Columns to add:** Date, Day, Event, Time, Driver out, Driver back, Notes.

**Recommended formatting:** freeze top row, color-band by family (light fills only), notes column for field changes or pickups.

**Sharing rule:** owner = team manager. Other drivers = editor. Spouses = viewer.

## 4. Fair lineup spreadsheet

**File:** `fair-lineup.gsheet`

**Audience:** rec coach managing fair playing time across a 12-kid roster.

**Body to paste:**

| Kid | Game 1 inn 1-3 | inn 4-5 | Game 2 inn 1-3 | inn 4-5 | Total mins | Notes |
|-----|---------------|---------|----------------|---------|------------|-------|

Add one row per kid. Track minutes by inning or quarter, not by appearances. Sum the column at the end. Aim for a max-min spread of 15 minutes across the team.

**Sharing rule:** coach + assistant. Not parents.

## 6. The hard parent email script (when a parent is upset)

**File:** `hard-parent-email-script.gdoc`

```
[Use only after thinking it through for 24 hours.]

Subject: Following up on Saturday

Hi [Parent Name],

I wanted to follow up on Saturday's game. I heard your concerns about [specific issue, no more than one sentence]. Here's what I'm thinking:

- [What I saw, neutral]
- [What I'm planning to do this week]
- [What I'd ask of you]

I'd be glad to talk in person if it's easier. I have ten minutes after Tuesday's practice.

Thanks,
Coach [first name]
```

**Rules:** never send before 24 hours have passed. Never copy other parents on it. Never address more than one issue per email. Always offer in-person follow-up.

## 8. Picture day shot list

**File:** `picture-day-shot-list.gdoc`

```
PARENT PHOTOGRAPHER (or volunteer parent) — 30-MINUTE LIST

Setup
□ Confirm time and location with team manager
□ Bring backup camera or charged phone
□ Check sun direction; position kids with sun behind photographer

Shots (in this order)
1. Full team — neutral pose, all looking at camera (3-5 frames)
2. Full team — playful pose / reaction shot (3 frames)
3. Coach + assistant + manager — together (2 frames)
4. Each kid solo — standing, smiling, no posed athletic stance (3 frames each)
5. Each kid solo — holding bat/ball/glove naturally (2 frames each)
6. Buddy shots — pairs of friends if requested (1-2 frames per pair)
7. Action — light jog, throw, kick (5-10 frames; for the website, not portraits)
8. Detail — gloves on bench, cleats, hat in grass (2-3 frames; for borders/headers)

Delivery
□ Upload to shared drive within 48 hours
□ Notify team chat with the link
□ Don't post individuals' faces publicly without parent permission
```

## How to ship these

1. For each markdown block above, create a Google Doc (for narrative templates) or Sheet (for the spreadsheets).
2. Paste in. Format minimally: H1 for the title, H2 for sections.
3. Set sharing to **Anyone with the link → Viewer**.
4. Copy the share link.
5. Send the agent the link list and we'll wire each one into `/resources/[name]/` on the site with a "Make a copy" button.
6. We'll also ship a PDF version of each (for download) generated from the same source. The PDF lives in `/public/` and the Google Doc link lives in the page body.

If a template needs a tweak before shipping, edit the markdown above first, then build the Doc.
