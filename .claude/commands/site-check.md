# /site-check

Run a full site readiness check before a major deploy or quarterly review.

Dispatches security-engineer, qa-human-tester, billing-finance, data-quality, and product-manager agents in parallel. Synthesizes findings into a single GO / NO-GO verdict.

Use before: shipping camps section, launching age-group pages, any infrastructure change, or quarterly review (next: September 2026).

Output: STATUS (GO / NO-GO / CONDITIONAL GO) + numbered blocker list + recommended next action.
